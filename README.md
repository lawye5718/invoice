# Invoice Processing System

This is a Streamlit-based invoice processing application that can handle various invoice formats including PDF and XML files.

## Features
- Process ZIP files containing invoices
- Extract data from PDF and XML invoices
- Match invoices with trip receipts
- Generate summary Excel reports
- Merge related PDF files
- Strict scope isolation to prevent cross-package matching errors

## Requirements
- Python 3.7+
- Streamlit
- pdfplumber
- pandas
- pypdf
